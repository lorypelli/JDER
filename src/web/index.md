# JDER

Editor di Diagrammi Entity-Relationship

## Il Tuo Editor ER Moderno

Nuova versione del software di [Alessandro Ballini](https://ballini.it/Software/ProgER), poi successivamente ripresa da [Gianvito Pio](https://gianvitopio.wordpress.com/jder), con una veste grafica più moderna e correzione di vari errori, riscritta completamente in Kotlin e Compose.

[Scarica JDER](https://github.com/LoryPelli/JDER/releases/download/JDER/JDER.jar)

## Caratteristiche Principali

### Interfaccia Moderna

Design pulito e intuitivo con supporto tema chiaro/scuro.

### Relazioni Avanzate

Gestione completa di relazioni N a N con entità associative.

### Esportazione Multipla

Salva i tuoi diagrammi in formato JSON o esportali come PNG.

### Note e Annotazioni

Aggiungi note e commenti ai tuoi diagrammi.

### Veloce e Reattivo

Scritto in Kotlin con Jetpack Compose per massime prestazioni.

### Facile da Usare

Interfaccia drag-and-drop intuitiva per creare diagrammi rapidamente.

## Tecnologie

- Kotlin
- Jetpack Compose
- Java
- ER Diagram
